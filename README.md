## About
使用 gin 开发興溢公司官网前后端分离API接口

## Feature
* JWT身份验证
* 文件上传与缩略图功能
* 提供图片验证码功能
* 发送短信功能
* 发送邮件功能
* 发送chatGPT功能
* 日志功能
* 实现跨域解决
* i18n多语言功能
* crontab定时任务功能
* WebSocket服务端

## Usage
```
# 执行
go run main.go
```

## Requires
go v1.19.3
gin v1.9.1
